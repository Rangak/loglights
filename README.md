# loglights


Extract Hyperledger Fabric Logfile Highlights


usage: loglights.py [-h] [-d DEPTH] [-p PRECISION] logfile


Example:

Create the logfile with timestamp option 

    # docker logs -t 11164e35484e >& ordererlogfile
    # loglights ordererlogfile
